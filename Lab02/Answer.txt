Section 10:
    Question:
        - If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
    Answer:
        No, Java does not allow this if both parameters are of the same type (e.g., both are String). 
        Java distinguishes overloaded constructors by the number, type, and order of parameters, not by the parameter names. Since DVD(String title) and DVD(String category) have the same signature (DVD(String)), it will cause a compile-time error.
